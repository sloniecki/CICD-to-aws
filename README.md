# CI/CD to aws project


## Overview
### This project is a CI/CD pipelinie that deploy simple 3-tier application to aws cloud
 Stack:
 Frontend: React 
 Backend: Python/flask
 Database: MySql
 CI/CD: jenkins
 Provisioning: Terraform
 Cloud:AWS
