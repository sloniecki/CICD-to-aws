# CI/CD to aws project


## Overview
### This project is a CI/CD pipelinie that deploy simple 3-tier application to aws cloud
 Stack:<br/>
 Frontend: React <br/>
 Backend: Python/flask<br/>
 Database: MySql<br/>
 CI/CD: jenkins<br/>
 Provisioning: Terraform<br/>
 Cloud:AWS<br/>
